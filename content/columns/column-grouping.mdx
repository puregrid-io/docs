Columns can be logically grouped. There is a special `ColumnGroup` type for this, it has a subset of column options and also a `children` array. A definition in the `columns` array can be either type.

```ts
const [columns, setColumns] = useState<Columns<Product>>([
  {
    // This is a ColumnGroup.
    key: 'productInfo',
    header: 'Product Information',
    children: [
      {
        // This is a Column.
        key: 'productName',
        header: 'Name',
        getValue: p => p.name,
        align: ColumnAlign.End,
      },
      {
        // This is a Column.
        key: 'productQuantity',
        header: 'Qty.',
        getValue: p => p.quantity,
        align: ColumnAlign.End,
      },
    ],
  },
]);
```
